# Nctl: Another nginxctl but with golang (WIP)

## Requirements:
- Go 1.19 or upper

## Install
### Archlinux (aur)
### Debian/Ubuntu
### From source
first you have to clone repo to your local machine
```bash
git clone https://github.com/aazerra/nctl
cd nctl
```
run build command
```bash
go build .
```
